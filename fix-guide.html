AI Visibility Fix Guide

Official guide from Hormuz AI • Built in South Africa

Follow these steps to make your Shopify store visible to ChatGPT, Claude, Perplexity and other AI agents.

Step 1: Upload robots.txt (Fastest fix)

You should already have downloaded this file from the audit.

  1. Go to Shopify Admin → Online Store → Themes → Actions → Edit code
  2. In the left sidebar, click "Add new file" under the root folder
  3. Name it exactly robots.txt
  4. Paste the content from the downloaded file and Save

Step 2: Add JSON-LD Structured Data (Most Important)

  1. Go to Shopify Admin → Online Store → Themes → Actions → Edit code
  2. Open the file theme.liquid
  3. Find the closing </head> tag
  4. Paste this code just above it:
<script type="application/ld+json">
{
  "@context": "https://schema.org/",
  "@type": "Product",
  "name": "{{ product.title }}",
  "image": "{{ product.featured_image | img_url: 'grande' }}",
  "description": "{{ product.description | strip_html }}",
  "brand": { "@type": "Brand", "name": "{{ shop.name }}" }
}
</script>

Step 3: Make Product Feed Public

  1. Go to Shopify Admin → Online Store → Preferences
  2. Turn OFF password protection (or whitelist AI crawlers)
  3. Go to Apps → Shopify Search & Discovery → make sure product recommendations are enabled
